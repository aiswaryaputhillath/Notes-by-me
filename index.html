HTML Tables

To organise the data(for structuring and layout of various components), Tables are used.

table tag is used to define a table. As a table consists of rows, for adding a row tr tag is used and to add data in each row, we use td tag. We could always nest these attributes and create a nested table

example of the table is:

item 1 item 2
item 3 item 4

It contains 3 parts : thead, tbody and tfoot

Thead contains the heading of the table data. Its represented by th

The tables can be done without using this 3 parts, but its best to use them as its useful when styling them seperatly. For more info check the MDN page
